Nucleus Segmentation with StarDist in 2D (DSB2018 data)

Authorship

This example is part of the Stardist - Object Detection with Star-convex Shapes described in the papers: Cell Detection with Star-convex Polygons. by Schmid et al and Star-convex Polyhedra for 3D Object Detection and Segmentation in Microscopy. by Weigert et al.

Details

This example is a 2D image of stained nuclei part of the DSB2018 dataset where several nuclei are overlaped, therefore, making it harder to split and segment without taking the risk of creating artifacts in the process.

For the full documentation and more detailed examples please have a look at the official StarDist project page.
